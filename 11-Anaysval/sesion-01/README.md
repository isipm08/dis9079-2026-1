### Herramientas y lenguajes

Se vieron herramientas básicas del curso:

### Processing

Lenguaje para crear imágenes y proyectos visuales con código. 

Sirve para hacer:

- gráficos interactivos  
- animaciones  
- visualización de datos  

### p5.js

Versión de Processing que funciona en el navegador.

Permite hacer proyectos interactivos sin instalar programas.

### Sistemas de comunicación e interacción

### Servidor

- Sistema que conecta dispositivos y permite compartir información entre ellos.

### Protocolos de comunicación

- Reglas que permiten que distintos sistemas se entiendan y envíen datos correctamente.

### Protocolos vistos en clase

### MQTT

- Protocolo liviano de mensajería basado en el sistema de “publicar y suscribirse”. Se usa mucho en IoT.

### HTTPS

- Protocolo seguro de la web basado en HTTP con cifrado TLS. Sirve para enviar datos de forma protegida por internet.

### OSC

- Protocolo de comunicación usado en música, arte interactivo y sistemas multimedia en tiempo real. Es rápido y de baja latencia.

### Tecnologías relacionadas

* TouchOSC: app para controlar y enviar datos mediante pantallas táctiles usando OSC.  

* Flutter: herramienta para crear aplicaciones móviles y web con interfaces interactivas.

### Conceptos clave

### IoT (Internet of Things)

Red de objetos conectados que pueden enviar y recibir datos mediante internet o redes locales.

### Semiótica

Estudio de cómo los signos (imágenes, palabras, interfaces) crean significado.

### Computación física

Uso de sensores y microcontroladores (como Arduino) para conectar el mundo digital con objetos físicos.
