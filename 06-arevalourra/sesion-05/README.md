solemne 1


En esta sesión estaba programada la solemne 1, pero finalmente utilizamos el tiempo para avanzar en nuestros proyectos de forma grupal. Esto nos permitió enfocarnos en el desarrollo práctico y resolver dudas directamente en clase.

En nuestro caso, el objetivo del proyecto es lograr la conexión entre dos placas Arduino Uno R4 WiFi, ya que inicialmente habíamos considerado utilizar una Raspberry Pi Pico 2 W, pero decidimos cambiarla para simplificar la implementación y trabajar con dos dispositivos del mismo tipo. Utilizando la plataforma Adafruit IO, buscamos enviar datos desde una placa y recibirlos en la otra para controlar un LED de forma remota.

Específicamente, buscamos encender y apagar un LED a través de internet, lo que implica comprender cómo funcionan las conexiones, el envío de datos y la comunicación entre dispositivos en un contexto de IoT.

Durante la clase avanzamos en la configuración de las placas, la conexión de los componentes y la implementación del código, aunque aún estamos ajustando algunos detalles para lograr una comunicación estable entre ambos dispositivos.
