## Enfoque 
Este curso aborda la interacción digital desde una perspectiva crítica, entendiendo la tecnología no solo como herramienta técnica, sino como un sistema cultural, político y comunicacional. Se busca comprender cómo los sistemas digitales median relaciones y construyen significado.


## Historia

El desarrollo de la interacción digital está vinculado al cruce entre diseño, arte y programación.

- Lauren McCarthy, Kyle McDonald, Roy Macdonald  
  Referentes contemporáneos en arte digital, código creativo e interacción.



## Herramientas y lenguajes

### Processing
Lenguaje de programación y entorno de desarrollo orientado a la creación visual. Basado en Java, permite desarrollar:

- Gráficos interactivos  
- Animaciones  
- Visualización de datos  

Se utiliza ampliamente en arte generativo y educación.

### p5.js
Versión de Processing adaptada a la web. Funciona en el navegador y facilita la creación de proyectos interactivos sin instalación.



## Interacción inalámbrica y sistemas

### Servidor
Un servidor permite la comunicación entre múltiples dispositivos dentro de una red, actuando como intermediario en el intercambio de información.


## Protocolos 

Los protocolos son reglas que permiten que distintos sistemas se comuniquen correctamente.

### MQTT (Message Queuing Telemetry Transport)
- Protocolo de mensajería ligero  
- Diseñado para redes inestables o de bajo ancho de banda  
- Funciona con modelo **publish/subscribe**  
- Usa un **broker** para distribuir mensajes  

Ideal para IoT y dispositivos con recursos limitados.



### HTTPS (HyperText Transfer Protocol Secure)
- Protocolo estándar de la web  
- Permite transferencia segura de datos  
- No está optimizado para tiempo real  



### OSC (Open Sound Control)
- Protocolo para comunicación multimedia  
- Alta velocidad y precisión  
- Usado en música, arte digital y performances en vivo  
- Menor latencia que MQTT  



## Tecnologías relacionadas

- TouchOSC  
  Aplicación para enviar y controlar datos mediante interfaces táctiles usando OSC.

- Flutter  
  Framework para desarrollar aplicaciones móviles e interfaces interactivas.


## Conceptos clave

### IoT (Internet of Things)
Red de objetos físicos conectados a internet mediante sensores y software.

Permite:
- Monitoreo en tiempo real  
- Automatización  
- Intercambio de datos entre dispositivos  



### Semiótica
Disciplina que estudia cómo los signos producen significado.

Analiza:
- Imágenes  
- Objetos  
- Lenguaje  
- Interfaces  

Es clave para diseñar interacciones comprensibles.



## Computación física

La interacción digital también ocurre en el mundo físico mediante sensores y dispositivos.

Ejemplos:
- Arduino  
- Interfaces tangibles  
- Instalaciones interactivas  

Esto permite conectar lo digital con lo material.



## Ideas clave

- La programación es una herramienta creativa  
- Los sistemas interactivos dependen de protocolos  
- El diseño digital implica interpretar signos (semiótica)  
- IoT conecta objetos y genera nuevas interacciones  
- La tecnología está influida por su contexto cultural y político
