## Protocolos 

1. **MQTT** (Message Queuing Telemetry Transport): Protocolo de transporte de mensajería, eficiente y diseñado para redes inestables o de bajo ancho de banda. Es un protocolo de comunicación, una manera de ponerse de acuerdo para transmitir los mensajes.
  - Utiliza un agente broker para gestionar las publicaciones (publish) y las suscripciones (suscribe)   
2. **HHTPS** (HyperText Transfer Protocol Secure): Protocolo estándar para la transferencia de documentos web, utilizado cuando se requiere seguridad y la carga de datos no es crítica en tiempo real.
3. **OSC** (Open Sound Control): Protocolo para el intercambio de información entre computadoras, sintetizadores de sonido y otros dispositivos multimedia. Se destaca por su alta velocidad y precisión, ideal para rendimiento en directo, a diferencia de la latencia que puede tener MQTT.

### ¿Qué es processing?

> lenguaje de programación de código abierto y un entorno de desarrollo integrado (IDE) diseñado para la creación visual, artes generativas, gráficos interactivos y educación artística. Basado en Java, permite a diseñadores y principiantes programar fácilmente animaciones, visualizaciones de datos y aplicaciones interactivas con pocas líneas de código
