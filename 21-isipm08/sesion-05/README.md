## solemne 1
Para esta clase con mi grupo decidimos llegar más temprano, para poder organizar de mejor manera los tiempos. Luego de muchos intentos fallidos de nuestra solemne, nos dirigimos al Lid para resolver nuestras dudas con el profe :).

Nuestra solemne al principio se enfocaba en enviar un mensaje desde un Arduino R4 WIFI hacia la Raspberry Pi Pico 2 W, finalmente mateo nos recomendó simplificarlo a dos placas Arduino, para que así se nos hiciera más fácil, usando también Adafruit IO para que una placa de Arduino enviara la información y otro la recibiera. Además nuestro objetivo principal era poder apagar y encender un Led conectado a un circuito en una protoboard. 

Actualmente estamos mejorando códigos e intentando establecer la conexión de ambas placas Arduino. 

Seguimos intentando!
