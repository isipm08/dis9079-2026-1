## Raspberry Pi Pico 2 W

+ La Raspberry Pi Pico 2 W es una placa orientada a proyectos electrónicos y de bajo costo, desarrollada por la Raspberry Pi Foundation, organización fundada en 2009 con el objetivo de democratizar el acceso a la computación

Principales características:

+ Chip RP2350
+ Doble núcleo
+ Conectividad Wi-Fi (802.11n)
+ Bluetooth 5.2
+ Programable en C, C++ y Python

+ Se trata de una plataforma eficiente para proyectos que requieren conectividad inalámbrica y cierto nivel de procesamiento

## Diferencias entre Arduino y Raspberry Pi

+ Aunque ambos dispositivos pueden utilizarse en proyectos similares, presentan diferencias importantes:

Arduino:
+ Microcontrolador
+ Orientado a tareas específicas
+ Programación en C++
+ Raspberry Pi:
+ Sistema más complejo
+ Capaz de ejecutar procesos más avanzados
+ Programación en Python o MicroPython

+ En este contexto, Arduino se utiliza para control directo de hardware, mientras que Raspberry Pi permite manejar procesos más complejos o múltiples tareas.
